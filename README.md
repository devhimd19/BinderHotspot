# BinderHotspot
Structure and sequence-based prediction of protein binding hotspots on Sortase A (S. aureus), a gram-positive AMR drug target, using microenvironment feature analysis.
This project adapts structure- and sequence-based feature engineering methods (developed for cysteine post-translational modification prediction) to a new problem: identifying candidate binding hotspot residues on Sortase A (SrtA), a cell-envelope-associated enzyme from Staphylococcus aureus and a recognized antimicrobial resistance (AMR) drug target. Using protein microenvironment features (contact-shell composition, secondary structure context, sequence motifs) and evolutionary conservation, residues are ranked by predicted hotspot propensity and validated against SrtA's known catalytic triad (His120, Cys184, Arg197). The goal is to demonstrate how existing structural bioinformatics methods can be extended toward molecular recognition problems relevant to computational protein binder design.
